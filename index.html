Tecnologias

HTML5 é a quinta versão da linguagem de marcação padrão para a criação de páginas da web.

CSS3 é a terceira versão do Cascading Style Sheets, utilizado para estilizar e formatar páginas da web escritas em HTML.

JavaScript é uma linguagem de programação de alto nível, utilizada principalmente para adicionar interatividade e dinamismo a páginas da web.

Sass é uma linguagem de extensão para CSS, que oferece recursos como variáveis, aninhamento e mixins.

ReactJS é uma biblioteca JavaScript de código aberto para construção de interfaces de usuário interativas e reativas, desenvolvida pelo Facebook.
